Test documentation for the ``1-my_list``
==========================================

The function ``print_sorted`` prints list items in ascending order

Examples:
---------
	>>> import sys
	>>> sys.path.append("..") #add parent
	>>> MyList = __import__("1-my_list").MyList
	>>>
	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	>>> print(my_list)
	[1, 4, 2, 3, 5]
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]
	>>> print(my_list)
	[1, 4, 2, 3, 5]
	>>>
	>>> my_list_2 = MyList()
	>>> my_list_2.append(-1)
	>>> my_list_2.append(-8)
	>>> my_list_2.append(-32)
	>>> my_list_2.append(-5)
	>>> print(my_list_2)
	[-1, -8, -32, -5]
	>>> my_list_2.print_sorted()
	[-32, -8, -5, -1]
	>>> print(my_list_2)
	[-1, -8, -32, -5]
	>>>
	>>> my_l = MyList()
	>>> print(my_l)
	[]
	>>> my_l.print_sorted()
	[]
	>>> print(my_l)
	[]
